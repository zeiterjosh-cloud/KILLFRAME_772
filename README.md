# KILLFRAME_772

KILLFRAME is a modular Three.js first-person shooter prototype with dedicated player, weapon, enemy AI, killframe logic, UI, and game-management systems.

## File Structure

- `index.html`
- `main.js`
- `player.js`
- `weapon.js`
- `enemy.js`
- `killframe.js`
- `ui.js`
- `gameManager.js`
- `styles.css`

## Run in Browser

From the project root directory:

```bash
npm install
npm run start
```

Open `http://localhost:8080` and click the game view to lock pointer and start aiming.

### Controls

- `W A S D`: move
- `Mouse`: look
- `Left Mouse Button`: shoot
